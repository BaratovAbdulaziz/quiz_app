# Roadmap

## Phase 1 — MVP (Q3 2026)

Core platform with PDF parsing, AI quiz generation, practice mode, exam mode, library, sharing, and authentication.

## Phase 2 — V1.1 (Q4 2026)

- Quiz analytics and progress tracking
- Spaced repetition review
- Bookmarking questions
- Keyboard shortcuts
- Dark mode refinements

## Phase 3 — V1.2 (Q1 2027)

- Public quiz gallery
- Bulk PDF upload
- Batch editing quizzes
- Advanced search across library
- Export quizzes (PDF, CSV)
- Improved reporting

## Phase 4 — V2 (Q2 2027)

- Team and classroom support
- Student progress dashboards for teachers
- Adaptive difficulty
- AI explanations for every answer
- API for third-party integrations
- Mobile apps (iOS, Android)

## Long-Term

- Community quiz marketplace
- AI-generated study guides from quiz content
- Offline-first mobile experience
- LTI integration with LMS platforms
- Gamification and streaks
