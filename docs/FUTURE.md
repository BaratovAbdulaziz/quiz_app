# Future

## V1.1

- **Quiz analytics** — per-quiz score history, question-level performance insights.
- **Spaced repetition** — algorithmic review scheduling using a modified SM-2 system.
- **Public quiz gallery** — browse and search publicly shared quizzes.
- **Bookmark questions** — save individual questions for focused review.
- **Keyboard shortcuts** — navigate the application without a mouse.
- **Dark mode refinements** — properly themed scrollbars, focus rings, and system UI elements.

## V1.2

- **Bulk PDF upload** — upload and process multiple PDFs simultaneously.
- **Batch editing** — edit, reorder, or delete multiple quiz questions at once.
- **Advanced search** — full-text search across all quizzes including question content.
- **Export quizzes** — export to PDF (printable) and CSV (spreadsheet).
- **Quiz cloning and editing** — duplicate a quiz and modify questions.
- **Improved reporting** — admin dashboard for question report moderation.

## V2

- **Team and classroom support** — create groups, invite members, share folders.
- **Teacher dashboards** — view student progress and performance across shared quizzes.
- **Adaptive difficulty** — AI adjusts question difficulty based on user performance.
- **AI explanations** — AI generates explanations for why each answer is correct or incorrect.
- **Public API** — allow third-party integrations.
- **Mobile apps** — native iOS and Android applications.
- **Offline mode** — download quizzes for offline practice.

## Long-Term Ideas

- **Community quiz marketplace** — users can publish and sell quiz packs.
- **AI study guides** — generate comprehensive study notes from quiz content.
- **LTI integration** — integrate with Canvas, Moodle, Blackboard.
- **Gamification** — streaks, badges, leaderboards for motivation.
- **Multilingual support** — full UI translation and quiz content translation.
- **AI tutoring** — conversational AI that explains concepts based on quiz content.
- **Audio quizzes** — listen to questions and answer verbally.
- **Question banks** — large shared question pools with random selection.
- **Print import** — import from printed documents via camera scan.
